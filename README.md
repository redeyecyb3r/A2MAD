# AI-Assisted Multi-Agent Application Development (A2MAD)

Project folder structure (Claude)

```
your-project/
├── .claude/
│   ├── CLAUDE.md        # Project-specific rules summary + cross-references
│   ├── agents/          # Persona definitions (Claude Code format)
│   ├── commands/        # Slash commands (e.g. phase-1-define)
│   ├── rules/           # Individual rule files (*.md)
│   └── templates/       # PRD, SAD, MR templates (shared)
|   └── settings.json    # Permissions, AAMAD_TARGET_RUNTIME env
├── project-context/
│   ├── 1.define/
│   ├── 2.build/
│   └── 3.deliver/
├── AGENTS.md
├── CHECKLIST.md
└── README.md
```
